1. Чому при роботі з файлами в Python критично важливо використовувати блоки try...except?

Відповідь студента 2 (Репін Данііл):
Використання блоків try...except дозволяє обробляти можливі помилки під час роботи з файлами.
Це запобігає аварійному завершенню програми і дає змогу вивести зрозуміле повідомлення про помилку.

2. Які режими відкриття файлів існують у Python і чим вони відрізняються?

Відповідь студента 3 (Спічаков Павло):
У Python існують такі основні режими відкриття файлів:
 - 'r' — читання (файл має існувати);
 - 'w' — запис (створює новий файл або перезаписує існуючий);
 - 'a' — дописування (інформація додається в кінець файлу);
 - 'x' — створення нового файлу (викликає помилку, якщо файл вже існує);
 - '+' — дозволяє як читання, так і запис.

3. Які існують основні методи читання даних із файлу в мові програмування Python?

Відповідь студента 4 (Мартиненко Олександр):
Основні методи читання файлів у Python:
 - read() — читає увесь вміст файлу як один рядок;
 - readline() — читає по одному рядку за виклик функції;
 - readlines() — повертає список рядків файлу;
Також файл можна читати через цикл for, що є ефективним та зручним способом обробки великих файлів.
